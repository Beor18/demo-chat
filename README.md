# DemoChat
Realizado por Fernando Lopez - 2017

Iniciamos la App:

  * Instalamos las dependencias con `mix deps.get`
  * creamos y migramos la base de datos `mix ecto.create && mix ecto.migrate`
  * Instalamos dependencias de Node `npm install`
  * Iniciamos Servidor Phoenix con `mix phoenix.server`

Visitamos [`localhost:4000`](http://localhost:4000) desde el navegador.
